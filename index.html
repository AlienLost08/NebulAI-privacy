Privacy Policy

NebulAI

NebulAI is a privacy-first browser extension that summarizes pages, chats with content, explains highlighted text, and analyzes images using your own AI API key.

Last updated: March 22, 2026

What We Collect

We do not collect or store personal data on our servers. Everything stays in your browser.

  • API keys are stored locally in chrome.storage.local.
  • Page content and images are processed only when you trigger an action.

How Your Data Flows

When you use NebulAI, your browser sends content directly to your chosen AI provider.

  • OpenAI API: api.openai.com
  • Google Gemini API: generativelanguage.googleapis.com
  • Anthropic API: api.anthropic.com

1. Overview

NebulAI is a browser extension that summarizes web pages, chats with page content, explains highlighted text, and analyzes images. All AI requests are made directly from your browser to the provider you choose, using your own API key.

2. Data We Do Not Collect

We do not collect, store, transmit, or sell any of the following:

  • API keys
  • Web page content, images, or queries you analyze
  • Browsing history or visited URLs
  • Personal identification data
  • Analytics, telemetry, or crash reports

3. Local Storage

NebulAI stores API keys and preferences locally in your browser via chrome.storage.local. This data stays on your device and is never sent to any server operated by us.

4. Third-Party API Communication

When you use NebulAI, your browser communicates directly with the AI provider you select. Their privacy policies apply to those requests:

  • OpenAI Privacy Policy
  • Google Privacy Policy
  • Anthropic Privacy Policy

5. Permissions

NebulAI uses the following permissions only to provide core features:

  • activeTab to read the current page when you trigger an action
  • storage to save API keys and preferences locally
  • scripting to extract page content and selected text
  • contextMenus to enable right-click explain actions
  • sidePanel to show the NebulAI interface

6. No Tracking or Ads

NebulAI contains no tracking scripts, advertising SDKs, or analytics libraries. We do not monetize user data.

7. Children’s Privacy

NebulAI is not intended for children under 13. We do not knowingly collect information from children.

8. Changes

We may update this policy over time. Any changes will be posted here with an updated date.
